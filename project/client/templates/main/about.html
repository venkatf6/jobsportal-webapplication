Program Details

As a future job aspirant i want to develop a python based application that will allow users to register,create skill-set based profile, to visualize,notify the possible job matches based on the area of interest.



How is it going to work?

 

 1.  Job aspirant will have access to the web portal.

   

 2. Job aspirant will fill all the required details and register to the service.

     

3. Once the user completes the registration he/she will be redirected to the profile builder.

     

4. Profile builder accepted list of core competencies from the end user.

   

 5. Job Matcher will try to match the skill-set with the available jobs and provide the match prediction.

   

 6. Job Matcher will invoke the notification module to communicate with the user through email,sms notifications periodically.
